# 爱为 IoT SaaS 平台 — 面试题总览

## 项目概述

基于 Spring WebFlux + Netty 从0到1构建企业级IoT SaaS平台，支撑海量异构设备的统一接入、管理与智能联动。涵盖响应式架构、多协议适配、多租户体系、规则引擎等核心模块。
