# AI智能识别与检索 - 面试问题准备

## 目录结构

- `01-系统架构类.md` - 整体架构设计、技术选型相关问题
- `02-AI模型与算法类.md` - 模型选择、训练、优化相关问题
- `03-图像识别类.md` - 图像处理、识别流程相关问题
- `04-检索与存储类.md` - 数据检索、向量存储、索引相关问题
- `05-性能优化类.md` - 性能调优、并发处理相关问题
- `06-综合与开放性追问.md` - 项目难点、收获、改进方向等

## 使用说明

每个文件按照 Q&A 格式组织，包含：
- 面试官可能提出的问题
- 推荐的回答思路
- 关键技术点和亮点
