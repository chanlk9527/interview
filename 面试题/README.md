## 使用建议

1. **优先级排序**：01-04（Java 基础）→ 08-09（数据库）→ 05-06（框架）→ 11-14（中间件与分布式）→ 其余
2. **结合项目**：每个知识点尽量关联到自己的项目经历中，面试时能做到"从项目引出原理"
3. **深度优先**：高频考点（并发、JVM、MySQL、Redis）要深入到源码级别
4. **查漏补缺**：配合 [面试准备指南](../面试准备指南.md) 一起使用
